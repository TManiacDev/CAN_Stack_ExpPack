ISOTP_E_PARAM_CONFIG 

API service called with invalid config.

When CanTp_Transmit is called for a none configured PDU identifier or with an identifier for a received PDU.

ISOTP_E_PARAM_ID 

API service called with invalid Pdu ID.

When CanTp_Transmit is called for a none configured PDU identifier or with an identifier for a received PDU.

ISOTP_E_PARAM_POINTER 

API service called with NULL pointer.

In case of this error, the API service shall return immediately without any further action, besides reporting this development error.

ISOTP_E_INIT_FAILED 

Module initialization has failed.

ISOTP_E_UNINIT 

API service used without initialization.

On any API call except CanTp_Init(), CanTp_GetVersionInfo() and CanTp_MainFunction() if CanTp is in state CANTP_OFF
