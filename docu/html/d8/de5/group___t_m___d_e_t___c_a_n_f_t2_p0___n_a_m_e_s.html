Development errors known by the CanFT2P0 module.

Enumerator
E_PARAM_CONFIG 

API service called with invalid config.

When CanFT2p0_Transmit is called for a none configured PDU identifier or with an identifier for a received PDU.

E_PARAM_ID 

API service called with invalid Pdu ID.

When CanFT2p0_Transmit is called for a none configured PDU identifier or with an identifier for a received PDU.

E_PARAM_POINTER 

API service called with NULL pointer.

In case of this error, the API service shall return immediately without any further action, besides reporting this development error.

E_INIT_FAILED 

Module initialization has failed.

E_UNINIT 

API service used without initialization.

On any API call except CanFT2p0_Init(), CanFT2p0_GetVersionInfo() and CanFT2p0_MainFunction() if CanFT2p0 is in state CANFT_OFF

E_INVALID_TX_ID 

Invalid Transmit Pdu Id used.

E_INVALID_RX_ID 

Invalid Receive Pdu Id used.
